# Parallel Architecture of the Modified L+S Model for Cardiac Cine MRI
This repository presents a CUDA-based GPU implementation of the **Modified Low-Rank plus Sparse (L+S) Model** for efficient cardiac cine MRI reconstruction. It contains optimized kernel code and sample cardiac cine datasets to support high-performance computing research in medical imaging.
